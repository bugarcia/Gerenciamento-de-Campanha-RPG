# Gerenciamento-de-Campanha-RPG
Gestor de campanhas TTRPG em Python. Centralize o controle de combate (HP/Iniciativa), bestiário, NPCs e construção de mundo em uma interface única e intuitiva. Com persistência em SQLite e visual dark, é a ferramenta definitiva para mestres que buscam agilidade na narrativa e organização total do enredo.
